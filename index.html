Metode 1

Simple Moving Average

Metode paling dasar yang menghitung rata-rata aritmatika dari sekumpulan data terakhir.

Rumus Matematis

SMAt = (Xt + Xt-1 + ... + Xt-k+1) / k

Keterangan Simbol:

  • SMAt : Nilai hasil smoothing
  • Xt : Data asli (aktual)
  • k : Window size (Jumlah periode)

Analisis Kritis: SMA

Keuntungan

  • Sangat Intuitif: Mudah dijelaskan kepada stakeholder non-teknis.
  • Filter Efektif: Sangat baik untuk menghilangkan noise pada data stabil.

Kekurangan

  • Lagging Effect: Sinyal perubahan tren baru muncul terlambat.
  • Ghosting Effect: Data ekstrem lama masih mempengaruhi rata-rata.
Metode Unggulan

Single Exp. Smoothing

Memberikan bobot (weight) eksponensial. Data terbaru dianggap jauh lebih penting.

Rumus Rekursif

St = αXt + (1-α)St-1

Keterangan Simbol:

  • St : Nilai smoothing saat ini
  • St-1 : Nilai smoothing sebelumnya
  • Xt : Data aktual saat ini
  • α : Parameter bobot (0 < α < 1)

Beyond Single Smoothing

Metode untuk data dengan Tren dan Musiman

Double Exp. (Holt's)

Lt = αYt + (1-α)(Lt-1 + Tt-1)
Tt = β(Lt - Lt-1) + (1-β)Tt-1
Lt: Level Tt: Trend

Triple (Holt-Winters)

St = γ(Yt-Lt) + (1-γ)St-m
Ft+k = Lt + kTt + St+k-m
St: Seasonal m: Musim

Analisis Kritis: Exponential Smoothing

Keunggulan

  • Weighted Priority: Data terbaru memiliki bobot lebih besar.
  • Efisiensi Memori: Tidak perlu menyimpan ribuan data sejarah.

Kelemahan

  • Sensitivitas Parameter: Pemilihan nilai α sangat krusial.
  • Versi Single Terbatas: Gagal memprediksi data tren naik tajam.

Matriks Keputusan

Karakteristik Data Metode Terbaik Alasan Teknis
Datar / Stasioner SMA Single Exp Tidak ada tren yang perlu dikejar.
Memiliki Tren Double Exp (Holt) Memiliki komponen slope untuk mengikuti tren.
Tren + Musiman Triple Exp (Holt-Winters) Menangkap pola berulang secara matematis.

Python Simulation

Simulasi eksekusi kode pandas.

1 Import & Load.
2 Hitung SMA.
3 Hitung EWM.
4 Visualisasi.
import pandas as pd
import matplotlib.pyplot as plt

# 1. Load Data
df = pd.read_csv('data.csv', parse_dates=['Date'], index_col='Date')

# 2. SMA Calculation
df['SMA'] = df['Close'].rolling(window=5).mean()

# 3. Exponential Calculation
df['EXP'] = df['Close'].ewm(alpha=0.2, adjust=False).mean()

# 4. Plot Result
plt.figure(); plt.plot(df); plt.show()
