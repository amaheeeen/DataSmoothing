Apa itu Time Series?

Rangkaian titik data yang diobservasi dan dicatat dalam urutan waktu yang spesifik (kronologis).

Karakteristik Utama:

  • Dependensi Waktu: Urutan (t) tidak boleh diacak.
  • Interval Tetap: Data diambil per jam, hari, atau bulan.
  • Autokorelasi: Data masa lalu mempengaruhi masa depan.

Anatomi Data

Bedah Komponen Time Series

Data Time Series (Yt) umumnya terdiri dari gabungan 4 komponen ini:

1. Trend (T)

Gerakan jangka panjang (naik/turun) yang konsisten.

2. Seasonal (S)

Fluktuasi berulang pada periode tetap (contoh: penjualan baju saat lebaran).

3. Cyclic (C)

Gelombang naik turun yang tidak tetap periodenya (contoh: siklus ekonomi/resesi).

4. Irregular (I)

Disebut juga Noise. Acak, tak terprediksi, dan mengganggu sinyal asli.

INTI PERMASALAHAN

Smoothing = Penghapusan Noise

Analogi Sinyal Radio

Data asli seringkali kasar ("kresek-kresek"). Tujuan smoothing bukan memanipulasi data, tapi membuang fluktuasi acak agar pola tren asli (Sinyal) terlihat jelas.

SensitifSangat Halus (Lag Tinggi)

Analisis Kritis: SMA

Keuntungan

  • Intuitif: Konsepnya sangat mudah dipahami oleh orang awam/stakeholder non-teknis.
  • Efektif untuk Data Stabil: Bekerja sangat baik pada data yang tidak memiliki tren tajam atau musiman kuat.

Kekurangan

  • Lagging Effect: Lambat merespon perubahan. Sinyal baru muncul setelah tren sebenarnya sudah terjadi.
  • Equal Weighting: Data 10 hari lalu dianggap sama pentingnya dengan data kemarin. Ini tidak realistis untuk prediksi.
  • Ghosting Effect: Nilai ekstrem lama masih mempengaruhi rata-rata meski sudah tidak relevan.
Metode Unggulan

Exponential Smoothing

Memberikan bobot (weight) berbeda. Data terbaru mendapat bobot terbesar, data lama mengecil secara eksponensial.

Analisis Kritis: Exponential Smoothing

Keuntungan

  • Weighted Priority: Sangat logis. Kejadian kemarin lebih relevan daripada kejadian setahun lalu.
  • Efisiensi Komputasi: Rumus rekursif. Hanya butuh data saat ini dan hasil smoothing sebelumnya. Tidak perlu menyimpan seluruh data sejarah (hemat memori).

Kekurangan

  • Sensitivitas Parameter: Pemilihan nilai α sangat krusial. Salah pilih bisa membuat model terlalu kasar atau terlalu lambat.
  • Versi Single Terbatas: Single Exponential Smoothing gagal memprediksi data yang memiliki tren naik/turun tajam (butuh Double/Holt).

Kapan Harus Menggunakan Apa?

Kondisi Data Rekomendasi Metode Alasan
Data Statis / Datar SMA atau Single Exp Tidak ada tren yang perlu dikejar. Rata-rata cukup baik.
Ada Tren (Naik/Turun) Double Exp (Holt's) SMA akan sangat lagging. Double Exp memiliki komponen slope untuk mengikuti tren.
Tren + Musiman Triple Exp (Holt-Winters) Satu-satunya metode smoothing klasik yang bisa menangkap pola berulang (musim).
Data Keuangan (Volatil) Exponential Harga kemarin lebih relevan daripada harga bulan lalu.

Implementasi Python

Menggunakan library pandas.

1

SMA: .rolling()

Fungsi .rolling(window=k) membuat "jendela" bergerak.
Note: k-1 data awal akan NaN.

2

Exp Smoothing: .ewm()

Menggunakan .ewm(alpha=...). Parameter adjust=False penting agar menggunakan rumus rekursif standar.

smoothing.py
import pandas as pd
import matplotlib.pyplot as plt

# 1. Load Data
df = pd.read_csv('data.csv')
df['Date'] = pd.to_datetime(df['Date'])
df.set_index('Date', inplace=True)

# 2. SMA Calculation
k = 5
df['SMA'] = df['Close'].rolling(window=k).mean()

# 3. Exponential Calculation
a = 0.2
df['EXP'] = df['Close'].ewm(alpha=a, adjust=False).mean()

# 4. Plot
plt.figure(figsize=(12,6))
plt.plot(df['Close'], label='Original')
plt.plot(df['SMA'], label='SMA', color='red')
plt.plot(df['EXP'], label='Exp', color='cyan')
plt.show()
