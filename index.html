Mengapa Kita Membutuhkan
Data Smoothing?

Masalah Utama

Data Time Series di dunia nyata tidak pernah mulus. Penuh dengan Noise dan Outlier.

Analogi Radio

Suara penyiar (Signal) sering tertutup kresek-kresek (Noise). Smoothing membuang suara kresek tersebut.

Komponen Data Time Series

Sebelum smoothing, pahami anatomi datanya.

1. Trend

Arah pergerakan jangka panjang (naik/turun). Ini adalah "Sinyal" utama.

2. Seasonality

Pola berulang pada interval tetap. Contoh: Penjualan baju naik tiap Lebaran.

3. Irregular / Noise

Fluktuasi acak. Inilah target utama yang ingin kita haluskan.

Metode 1

Simple Moving Average (SMA)

Cara Kerja

Mengambil rata-rata dari sejumlah k titik data terakhir.

SMA(3) = (Jan + Feb + Mar) / 3

Kelemahan Fatal

  • Lagging: Lambat merespon perubahan tren.
  • Bobot Sama: Data lama dianggap sama pentingnya dengan data terbaru.

Simulasi Lagging

3

Geser ke kanan untuk melihat efek Lag

Metode Unggulan

Exponential Smoothing

Perbaikan dari SMA.

Konsep Utama: Memberikan bobot (weight) berbeda. Data terbaru bobot terbesar.

Parameter Alpha (α)

  • ≈ 1 Sangat reaktif (Kurva kasar).
  • ≈ 0 Sangat halus (Prioritas sejarah).

Variasi Metode:

  • Single: Data tanpa tren.
  • Double (Holt): Data dengan tren.
  • Triple (Holt-Winters): Tren + Musiman.

Argumen: Lebih akurat menangkap perubahan tren dibanding SMA.

Simulasi Parameter Alpha

0.2
Smooth (History) Kasar (Reaktif)

Evaluasi & Pengukuran Error

Bagaimana kita tahu hasil smoothing kita berkualitas?

Metrik Statistik

MSE

Mean Squared Error

Menghukum kesalahan besar lebih berat. Bagus untuk melihat variansi.

Direkomendasikan
Metrik Bisnis

MAPE

Mean Absolute Percentage Error

Rata-rata persentase kesalahan. Jika MAPE 5%, berarti prediksi meleset 5%.

Implementation

Implementasi Python

Berikut adalah cara menerapkan SMA dan Exponential Smoothing menggunakan library pandas.

1. Load Data

Pastikan kolom tanggal di-parse menggunakan parse_dates dan dijadikan index.

2. SMA (Rolling)

Gunakan fungsi .rolling(window=k).mean(). Sangat sederhana namun efektif untuk tren kasar.

3. Exp Smoothing (EWM)

Gunakan .ewm(alpha=...).mean(). Fungsi Exponential Weighted Functions bawaan Pandas.

smoothing_analysis.py 
import pandas as pd
import matplotlib.pyplot as plt

# 1. Persiapan Data (Load CSV)
df = pd.read_csv('saham_bbri.csv', parse_dates=['Date'])
df.set_index('Date', inplace=True)

# 2. Simple Moving Average (SMA)
# Window 30 hari untuk melihat tren bulanan
df['SMA_30'] = df['Close'].rolling(window=30).mean()

# 3. Exponential Smoothing (Pandas EWM)
# Alpha 0.2 (Halus) vs 0.8 (Reaktif)
df['EXP_02'] = df['Close'].ewm(alpha=0.2, adjust=False).mean()

# 4. Visualisasi Hasil
plt.figure(figsize=(12,6))
plt.plot(df['Close'], label='Original Data', alpha=0.5)
plt.plot(df['SMA_30'], label='SMA (30)', color='red')
plt.plot(df['EXP_02'], label='Exp Smoothing (0.2)', color='cyan')

plt.legend()
plt.title('Perbandingan Metode Smoothing')
plt.show()
