Definisi
Time Series

Sekumpulan titik data yang direkam pada interval waktu tertentu secara berurutan. Berbeda dengan Cross-Sectional data.

Hukum Wajib:

  • Order Matters: Urutan ($t_1, t_2, ...$) Sakral.
  • Autocorrelation: Masa lalu mempengaruhi masa depan.

Analisis Kritis: SMA

Keuntungan

Intuitif & Filter efektif untuk data stabil.

Kelemahan Fatal

  • Lagging Effect: Respon lambat.
  • Ghosting: Data ekstrem lama menghantui.
  • Outlier Sensitive: 1 data error merusak rata-rata.

Solusi: Moving Median

"Jika data Anda banyak 'sampah' (spikes/glitch), jangan pakai Rata-rata (Mean). Pakai Nilai Tengah (Median)."

Data: [10, 12, 900, 11, 13]
Mean = 189.2 (Rusak!)
Median = 12 (Aman)

Advanced Methods

Adaptasi untuk pola Tren dan Musiman

Double (Holt)

Lt = Level (Rata-rata)

Tt = Trend (Kemiringan)

Digunakan jika data memiliki tren naik/turun yang jelas.

Triple (Winters)

St = Seasonal Component

m = Panjang Musim

Digunakan jika data berulang (misal: penjualan es krim tiap musim panas).

Analisis Kritis: Exponential

Pro: Efisiensi Memori

Tidak perlu menyimpan database sejarah panjang. Rekursif.

Con: Dilema Alpha

Alpha kecil = Lagging. Alpha besar = Noisy. Susah ditebak.

Solusi Rigorous: Solver Optimization

Jangan tebak Alpha! Gunakan algoritma Grid Search (coba 0.01 - 0.99) untuk mencari Alpha dengan Error terkecil.

Evaluasi Kuantitatif

"Don't Trust Your Eyes"

1. MSE (Mean Squared Error)

Menghukum kesalahan besar lebih berat. Standar industri.

Σ(Actual - Forecast)² / n

2. MAPE (Percentage Error)

Kesalahan dalam persen. Mudah dipahami bisnis.

Σ(|Error| / Actual) / n * 100%

Python Simulation

Simulasi eksekusi kode pandas.

1 Load Data
2 Hitung SMA
3 Hitung EWM (Exp)
4 Plot Grafik
import pandas as pd
import matplotlib.pyplot as plt
# 1. Load
df = pd.read_csv('data.csv', index_col='Date')
# 2. SMA
df['SMA'] = df['Close'].rolling(window=5).mean()
# 3. Exp
df['EXP'] = df['Close'].ewm(alpha=0.2).mean()
# 4. Plot
df.plot(); plt.show()
