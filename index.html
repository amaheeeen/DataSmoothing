Definisi Teknis

Apa itu Data Smoothing?

Metode statistik untuk menghilangkan variasi acak (noise) dari dataset. Tujuannya adalah memperhalus kurva agar pola fundamental (Trend/Seasonal) terlihat jelas.

Visualisasi

Mempermudah otak manusia melihat pola arah di tengah grafik yang "gerigi".

Prediksi

Data bersih menghasilkan model forecasting yang lebih akurat (tidak overfitting ke noise).

Metode 1

Simple Moving Average (SMA)

Metode paling dasar yang menghitung rata-rata aritmatika dari k data terakhir. Sering disebut "Jendela Berjalan".

SMAt = Σ Xi / k

Bedah Logika

  • Bobot Setara: Data H-1 dianggap sama pentingnya dengan data H-5. Ini kelemahan utamanya (naif).
  • Filter Efektif: Sangat bagus untuk membuang noise frekuensi tinggi pada data yang trennya datar.

Metode 2

Weighted Moving Average (WMA)

Memperbaiki kenaifan SMA dengan memberikan Bobot (w) berbeda. Data terbaru diberi bobot terbesar.

WMAt = Σ (wi ċ Xi) / Σ wi

Logika Pasar: Harga saham kemarin lebih relevan memprediksi hari ini dibanding harga bulan lalu. Maka, bobot kemarin harus lebih besar.

Metode 3

Single Exponential Smoothing

Solusi elegan untuk masalah bobot. Menggunakan satu parameter Alpha (α) untuk menurunkan bobot data lama secara eksponensial (melengkung).

St = αXt + (1-α)St-1

Peran Alpha (α)

Alpha Kecil (0.1)
Sangat halus, tapi lag parah. Mengutamakan sejarah panjang.
Alpha Besar (0.9)
Sangat responsif, tapi masih kasar (noisy). Mengutamakan data baru.

Advanced Methods: Beyond Single Level

Untuk data yang memiliki pola Tren (Naik/Turun) dan Musiman.

Double Exp (Holt's Method)

Menambahkan persamaan kedua untuk memuluskan Tren (Slope/Kemiringan).

Level: Lt = αYt + (1-α)(Lt-1 + Tt-1)
Trend: Tt = β(Lt - Lt-1) + (1-β)Tt-1
Forecast: Yt+h = Lt + hTt
Simulation: Linear Trend

Triple Exp (Holt-Winters)

Metode terlengkap. Menambahkan persamaan ketiga untuk pola Musiman (Seasonal).

Season: St = γ(Yt-Lt) + (1-γ)St-m
Forecast: Yt+h = Lt + hTt + St+h-m
*Formula disederhanakan

Analisis Kritis & Trade-off

Keunggulan Exponential

  • Efisien: Hanya butuh data terakhir & nilai forecast terakhir.
  • Adaptif: Bisa diatur responsivitasnya lewat Alpha.

Kelemahan

  • Lagging: Masih ada keterlambatan respon vs data asli.
  • Parameter: Sulit menentukan α yang optimal (butuh Trial/Error).

Solusi Rigorous: Solver Optimization

Jangan tebak Alpha! Gunakan algoritma Grid Search (coba 0.01 - 0.99) untuk mencari Alpha dengan Error terkecil secara otomatis.

MSE Formula (Hukuman Berat)
Σ(Aktual - Forecast)2 / n

Menguadratkan error, sehingga kesalahan besar sangat "menyakitkan" bagi skor.

MAPE Formula (Persentase)
Σ(|Error| / Aktual) / n

Rata-rata persentase kesalahan. Lebih mudah dipahami oleh bisnis.

Python Code

Simulasi implementasi menggunakan library Pandas.

1 Load Data
2 Hitung SMA
3 Hitung EWM (Exp)
4 Plot Grafik
import pandas as pd
import matplotlib.pyplot as plt
# 1. Load Data
df = pd.read_csv('data.csv', index_col='Date')
# 2. SMA (Window=5)
df['SMA'] = df['Close'].rolling(window=5).mean()
# 3. Exponential (Alpha=0.2)
df['EXP'] = df['Close'].ewm(alpha=0.2).mean()
# 4. Visualization
df.plot(); plt.show()
