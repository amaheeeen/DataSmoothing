Apa itu Time Series?

Data yang direkam dalam urutan waktu kronologis. Urutan ($t$) dan interval waktu adalah kunci utama.

  • Urutan tidak boleh diacak.
  • Interval waktu harus konsisten.

Smoothing = Membuang Noise

"Data mentah seperti radio saat badai. Suara 'kresek' (Noise) menutupi suara penyiar (Signal). Smoothing adalah filter untuk menjernihkan suara tersebut."

Metode 1

Simple Moving Average

Rata-rata aritmatika dari $k$ data terakhir.

SMAt = (Xt + ... + Xt-k+1) / k

Simbol:

  • $X_t$: Data asli
  • $k$: Window size

Simulasi Lagging

Window ($k$)3

Kelemahan SMA

  • Lagging: Telat respon tren.
  • Equal Weight: Data lama = Data baru.
  • Sensitif Outlier: Satu error merusak rata-rata.

Solusi: Median

Moving Median lebih tahan terhadap data ekstrem.

Metode 2

Single Exponential

Bobot menurun secara eksponensial. Data baru prioritas utama.

St = αXt + (1-α)St-1

$\alpha \approx 1$: Reaktif (Ikut fluktuasi)

$\alpha \approx 0$: Smooth (Ingat sejarah)

Simulasi Alpha

Alpha ($\alpha$)0.2

Beyond Single Smoothing

Double Exp (Holt)

Untuk data dengan Tren.

Menambah persamaan Slope (Kemiringan).

Triple Exp (Winters)

Untuk data dengan Musiman.

Menambah persamaan Seasonal.

Evaluasi Akurasi

"Visual bisa menipu. Kita butuh angka error."

MSE (Mean Squared Error): Menghukum outlier berat.
MAPE (Percentage Error): Persentase meleset.

Python Auto-Opt

Simulasi optimasi parameter otomatis dengan statsmodels.

from statsmodels.tsa.api import SimpleExpSmoothing


# 1. Initialize Model


model = SimpleExpSmoothing(data)


# 2. Auto-Optimize (Grid Search)


fit_model = model.fit(optimized=True)


# 3. Result


print(f"Best Alpha: {fit_model.params['smoothing_level']}")
