Definisi
Time Series

Rangkaian titik data yang direkam pada interval waktu tertentu secara berurutan. Urutan adalah kunci.

Hukum Wajib:

  • Urutan ($t$): Tidak boleh diacak.
  • Interval: Konsisten (harian, jam, dll).
  • Autokorelasi: $X_t$ bergantung pada $X_{t-1}$.

Metode 1

Simple Moving Average

Metode dasar. Menghitung rata-rata aritmatika dari sekumpulan data terakhir. Demokrasi penuh: semua data sama penting.

SMAt = (Xt + ... + Xt-k+1) / k

Legenda:

  • Xt : Data asli pada waktu t
  • k : Window size (Jumlah periode)

Kelemahan Fatal SMA

  • 1. Lagging: Terlambat merespon tren baru.
  • 2. Equal Weight: Data lama dianggap sama pentingnya dengan data baru.
  • 3. Sensitivitas Outlier: Satu data error (glitch) akan merusak rata-rata.

Solusi Alternatif: Moving Median

Median lebih robust (tahan banting) terhadap nilai ekstrem dibandingkan Mean (Rata-rata). Lihat simulasi di samping.

Metode 2 (Unggulan)

Single Exp. Smoothing

Solusi cerdas dengan satu parameter ($\alpha$). Data terbaru mendapat prioritas tertinggi.

St = αXt + (1-α)St-1

Legenda:

  • St : Nilai smoothing saat ini
  • St-1 : Nilai smoothing sebelumnya
  • Xt : Data asli saat ini
  • α : Parameter bobot (0 < α < 1)

Beyond Single Smoothing

Metode lanjutan untuk data kompleks (Tren & Musim).

Double Exp. (Holt)

Untuk data dengan Tren. Menambah persamaan slope.

Lt = αYt + (1-α)(Lt-1 + Tt-1)
Tt = β(Lt - Lt-1) + (1-β)Tt-1

Triple Exp. (Winters)

Untuk data dengan Musiman. Menambah persamaan seasonal.

St = γ(Yt-Lt) + (1-γ)St-m
Forecast = L + T + S

Python Auto-Opt

Menggunakan statsmodels untuk mencari parameter terbaik secara otomatis (Grid Search).

1. Initialize Model (SimpleExp).
2. Fit Model (Optimized=True).
3. Print Best Alpha.
4. Plotting Result.
from statsmodels.tsa.api import SimpleExpSmoothing
# 1. Initialize Model
model = SimpleExpSmoothing(data)
# 2. Auto-Optimize (Grid Search)
fit_model = model.fit(optimized=True)
# 3. Best Alpha Found
print(f"Best Alpha: {fit_model.params['smoothing_level']}")
# 4. Forecast
fcast = fit_model.forecast(3)
