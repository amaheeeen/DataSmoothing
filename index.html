Apa itu Time Series?

Rangkaian titik data yang diobservasi dan dicatat dalam urutan waktu yang spesifik (kronologis).

Karakteristik Utama:

  • Dependensi Waktu: Urutan (t) tidak boleh diacak.
  • Interval Tetap: Data diambil per jam/hari/bulan.
  • Autokorelasi: Data masa lalu mempengaruhi masa depan.

Bedah Komponen Time Series

Data Time Series (Yt) terdiri dari 4 komponen:

1. Trend (T)

Gerakan jangka panjang (naik/turun) yang konsisten.

2. Seasonal (S)

Fluktuasi berulang pada periode tetap (musiman).

3. Cyclic (C)

Gelombang naik turun yang tidak tetap periodenya.

4. Irregular (I)

Noise. Acak, tak terprediksi, dan mengganggu sinyal asli.

INTI PERMASALAHAN

Smoothing = Penghapusan Noise

Analogi Sinyal Radio

Data asli seringkali kasar ("kresek-kresek"). Tujuan smoothing bukan memanipulasi data, tapi membuang fluktuasi acak agar pola tren asli (Sinyal) terlihat jelas.

Metode 1

Simple Moving Average (SMA)

Metode paling dasar. Menghitung rata-rata aritmatika dari k titik data terakhir.

Rumus Dasar

SMAt = (Xt + Xt-1 + ... + Xt-k+1) / k

Keterangan Simbol:

  • SMAt : Nilai hasil smoothing pada waktu t
  • Xt : Data asli (aktual) pada waktu t
  • k : Window size (Jumlah periode rata-rata)

Metode Unggulan

Single Exponential Smoothing

Memberikan bobot (weight) berbeda. Data terbaru mendapat bobot terbesar.

Rumus Dasar

St = αXt + (1 - α)St-1

Keterangan Simbol:

  • St : Nilai hasil smoothing saat ini
  • St-1 : Nilai hasil smoothing sebelumnya
  • Xt : Data asli (aktual) saat ini
  • α : Parameter smoothing (0 < α < 1)

Beyond Single Smoothing

Metode untuk data dengan Tren dan Musiman

Double Exp. (Holt's)

Digunakan ketika data memiliki Tren (naik/turun).

Lt = αYt + (1-α)(Lt-1 + Tt-1)
Tt = β(Lt - Lt-1) + (1-β)Tt-1

Lt: Level (Nilai dasar) Tt: Trend (Kemiringan) α: Smoothing Level β: Smoothing Trend

Triple Exp. (Holt-Winters)

Untuk data dengan Tren + Musiman.

Lt = ... (Level)
Tt = ... (Trend)
St = γ(Yt-Lt) + (1-γ)St-m

St: Seasonal Component γ: Smoothing Seasonal m: Panjang Musim

Analisis Kritis: Exponential Smoothing

Keuntungan

  • Weighted Priority: Sangat logis. Kejadian kemarin lebih relevan daripada kejadian setahun lalu.
  • Efisiensi Komputasi: Rumus rekursif. Hanya butuh data saat ini dan hasil sebelumnya.

Kekurangan

  • Sensitivitas Parameter: Pemilihan nilai α sangat krusial.
  • Versi Single Terbatas: Gagal memprediksi data yang memiliki tren naik/turun tajam.

Kapan Harus Menggunakan Apa?

Kondisi Data Rekomendasi Metode Alasan
Data Statis / Datar SMA Single Exp Tidak ada tren yang perlu dikejar. Rata-rata cukup baik.
Ada Tren (Naik/Turun) Double Exp (Holt's) SMA akan sangat lagging. Double Exp memiliki komponen slope.
Tren + Musiman Triple Exp (Holt-Winters) Satu-satunya metode smoothing klasik yang menangkap pola berulang.

Python Simulation

Simulasi eksekusi kode pandas untuk melakukan smoothing data saham secara real-time.

1 Import library dan load data.
2 Hitung SMA (Rolling Window).
3 Hitung Exponential Smoothing (EWM).
4 Visualisasi (Show Plot Window).
import pandas as pd
import matplotlib.pyplot as plt

# 1. Load Data
df = pd.read_csv('data.csv', parse_dates=['Date'], index_col='Date')

# 2. SMA Calculation
df['SMA'] = df['Close'].rolling(window=5).mean()

# 3. Exponential Calculation
df['EXP'] = df['Close'].ewm(alpha=0.2, adjust=False).mean()

# 4. Plot Result
plt.figure(figsize=(10,6)); plt.plot(df); plt.show()
