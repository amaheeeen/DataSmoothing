Definisi Time Series

Time Series adalah sekumpulan titik data yang direkam pada interval waktu tertentu secara berurutan dan kronologis. Berbeda dengan data Cross-Sectional yang diambil pada satu waktu, Time Series sangat bergantung pada urutan.

Mengapa Smoothing Penting?
Data Time Series seringkali "kotor" dan fluktuatif. Smoothing membantu kita melihat gambaran besar (Trend) tanpa terganggu oleh fluktuasi harian.

Hukum Wajib:

  • Order Matters: Urutan (t1, t2) Sakral.
  • Fixed Interval: Jarak waktu konsisten (harian, jam, detik).

Metode 1

Simple Moving Average (SMA)

SMA adalah metode smoothing paling dasar. Cara kerjanya adalah mengambil rata-rata aritmatika dari sejumlah data terakhir (disebut window k).

Karakteristik: Memberikan bobot yang SAMA rata untuk setiap data dalam window. Data kemarin dianggap sama pentingnya dengan data 5 hari lalu.

Metode 2

Weighted Moving Average (WMA)

Mengapa SMA saja tidak cukup?
Kelemahan SMA adalah menganggap data lama sama pentingnya dengan data baru. Padahal dalam realita (misal: harga saham), harga kemarin lebih relevan memprediksi hari ini dibanding harga minggu lalu.

Solusi WMA:
Kita memberikan Bobot (w) yang berbeda. Data terbaru diberi bobot terbesar, data lama diberi bobot terkecil.

WMAt = Σ (wi ċ Xi) / Σ wi

Single Exponential Smoothing

Jika WMA butuh banyak bobot manual, Exponential Smoothing menyelesaikannya dengan satu parameter saja: Alpha (α).

Bobot menurun secara eksponensial otomatis. Sangat efisien memori karena bersifat rekursif.

Advanced Exponential Methods

Single Exp hanya untuk data datar. Bagaimana jika ada Tren/Musim?

Double (Holt)

Digunakan jika data memiliki Tren (naik/turun). Menambahkan persamaan kedua untuk memuluskan kemiringan (slope).

Lt = Level (Rata-rata)

Tt = Trend (Kemiringan)

Triple (Holt-Winters)

Digunakan jika data memiliki Tren + Musiman. Metode paling kompleks dan lengkap.

St = Seasonal Component

m = Panjang Musim (mis: 12 bulan)

Analisis Kritis & Trade-off

Keunggulan Exponential

  • Efisien: Hanya butuh data terakhir & nilai forecast terakhir.
  • Adaptif: Bisa diatur responsivitasnya lewat Alpha.

Kelemahan

  • Lagging: Masih ada keterlambatan respon vs data asli.
  • Parameter: Sulit menentukan α yang optimal (butuh Trial/Error).

Solusi Rigorous: Solver Optimization

Jangan tebak Alpha! Gunakan algoritma Grid Search (coba 0.01 - 0.99) untuk mencari Alpha dengan Error terkecil secara otomatis.

Evaluasi Akurasi (MSE & MAPE)

Python Code

Simulasi implementasi menggunakan library Pandas.

1 Load Data
2 Hitung SMA
3 Hitung EWM (Exp)
4 Plot Grafik
import pandas as pd
import matplotlib.pyplot as plt
# 1. Load Data
df = pd.read_csv('data.csv', index_col='Date')
# 2. SMA (Window=5)
df['SMA'] = df['Close'].rolling(window=5).mean()
# 3. Exponential (Alpha=0.2)
df['EXP'] = df['Close'].ewm(alpha=0.2).mean()
# 4. Visualization
df.plot(); plt.show()
